# Generador de Tarjetas de Visita

Una pequeña aplicación web para crear tarjetas de visita (business cards) elegantes en
segundos. Escribes tus datos —nombre, cargo, empresa, teléfono, correo y sitio web— y la
tarjeta se previsualiza al instante. Cuando te gusta cómo se ve, puedes **imprimirla** o
**descargarla como imagen PNG**.

Todo funciona sin internet y sin instalar nada: es un único archivo HTML con el CSS y el
JavaScript incluidos dentro.

## ¿Cómo se usa?

1. Abre el archivo `index.html` con doble clic (se abre en tu navegador).
2. Rellena los campos del formulario de la izquierda.
3. Mira la tarjeta actualizarse en vivo a la derecha.
4. Pulsa **🖨️ Imprimir** para enviarla a la impresora, o **⬇️ Descargar imagen** para
   guardarla como PNG en tu computadora.

## Características

- Previsualización en tiempo real mientras escribes.
- Diseño moderno con colores corporativos (azul/gris) y buen contraste.
- Descarga como imagen PNG generada con `canvas`, sin librerías externas.
- Opción de impresión directa desde el navegador.
- Sin CDNs, sin dependencias, sin conexión a internet.

## Ángulo de monetización

Se puede ofrecer como servicio de diseño exprés: plantillas premium de pago (más estilos,
logotipos, códigos QR) o una versión "pro" que exporta en alta resolución y formatos para
imprenta. También sirve como gancho para captar clientes de un estudio de diseño que luego
contraten identidad de marca completa.
